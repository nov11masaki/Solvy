�� 起動方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. セットアップ:
   $ chmod +x setup.sh && ./setup.sh

2. 環境変数設定:
   $ echo "OPENAI_API_KEY=sk-xxxx" > backend/.env

3. 開発サーバー起動:
   $ npm run dev

4. ブラウザで確認:
   • フロントエンド: http://localhost:5173
   • バックエンド: http://localhost:5000
